# Overview

Proxed.AI is a platform for building secure, scalable, and compliant API for AI wrappers. It acts as a secure proxy between your applications and AI providers like OpenAI, Anthropic, and Google AI.

![Proxed.AI Logo](/opengraph-image.png)

## Features

- **Secure**: Built with enterprise-grade security including DeviceCheck verification and partial API key management
- **Multi-Provider Support**: Seamlessly work with OpenAI, Anthropic, and Google AI (Gemini) models
- **Structured Responses**: Define schemas to ensure consistent, type-safe outputs from AI models
- **Scalable**: Designed to handle production workloads with built-in rate limiting and monitoring
- **Compliant**: Adheres to the latest security standards with comprehensive audit trails

## Supported AI Providers

- **OpenAI**: GPT-4.1, GPT-4o, GPT-4o-mini, and other models
- **Anthropic**: Claude 4 family (Opus, Sonnet)
- **Google AI**: Gemini 2.5, 2.0, and 1.5 series models

## Self-Hosting

Proxed.AI can be self-hosted.

To self-host Proxed.AI, you need to have accounts for the following services:

- [Supabase](https://supabase.com)
- [Vercel](https://vercel.com)
- [GitHub](https://github.com)
