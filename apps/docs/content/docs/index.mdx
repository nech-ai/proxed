# Overview

proxed.ai is a platform for building secure, scalable, and compliant API for AI wrappers.

![proxed.ai Logo](/opengraph-image.png)

## Features

- **Secure**: proxed.ai is built with security in mind.
- **Scalable**: proxed.ai is designed to be scalable.
- **Compliant**: proxed.ai is compliant with the latest security standards.

## Self-Hosting

proxed.ai can be self-hosted.

To self-host proxed.ai, you need to have accounts for the following services:

- [Supabase](https://supabase.com)
- [Vercel](https://vercel.com)
- [GitHub](https://github.com)
